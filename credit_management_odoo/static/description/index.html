Gestión de Crédito para Clientes

Controla los límites de crédito desde Contactos e Invoicing

Este módulo permite gestionar el crédito de los clientes directamente desde el módulo de Contactos, integrado con Contabilidad. Se generan alertas automáticas si se supera el límite de crédito configurado para un cliente.

  • Campo de límite de crédito por cliente
  • Validaciones durante la facturación si se excede el crédito
  • Soporte multicurrency para cálculos de crédito
  • Reporte general de clientes con crédito y su saldo actual
  • Permisos basados en roles para modificar o visualizar información de crédito
