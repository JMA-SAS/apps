Gestion de Credito para Clientes

Controla los limites de credito desde Contactos e Invoicing

Este modulo permite gestionar el credito de los clientes directamente desde el modulo de Contactos, integrado con Contabilidad. Se generan alertas automaticas si se supera el limite de credito configurado para un cliente.

  • Campo de limite de credito por cliente
  • Validaciones durante la facturacion si se excede el credito
  • Soporte multicurrency para calculos de credito
  • Reporte general de clientes con credito y su saldo actual
  • Permisos basados en roles para modificar o visualizar informacion de credito
