Exportador a FTP para Odoo

Inventario y Ventas Diarias en CSV

Este módulo permite exportar automáticamente información diaria del inventario y las ventas a un servidor FTP en formato CSV. La exportación se realiza de forma automatizada mediante acciones programadas en Odoo.

  • Exportación de inventario por producto
  • Exportación de ventas diarias por producto
  • Generación automática de archivos CSV
  • Subida directa al servidor FTP configurado
  • Automatización con acciones programadas

Archivos generados

  • inventario_YYYYMMDD.csv
  • ventas_YYYYMMDD.csv

Configuración del FTP

Para configurar el servidor FTP, edita el archivo ftp_export.py con tus credenciales:

ftp_host = 'ftp.tuservidor.com'
ftp_user = 'tu_usuario'
ftp_pass = 'tu_contraseña'
ruta_remota = '/ruta/ftp/'

Instalación

  1. Copia el módulo en la carpeta addons de tu instancia Odoo.
  2. Instálalo desde el panel de aplicaciones (modo desarrollador).
  3. Crea una acción programada para automatizar la exportación diaria.
