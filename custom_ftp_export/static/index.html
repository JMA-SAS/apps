Exportador a FTP para Odoo

Este modulo permite exportar diariamente:

  • Inventario por producto
  • Ventas del dia por producto

Características

  • Archivos CSV generados automaticamente
  • Subida directa a un servidor FTP
  • Automatización mediante acciones programadas

Archivos generados

  • inventario_YYYYMMDD.csv
  • ventas_YYYYMMDD.csv

Configuración del FTP

Edita el archivo ftp_export.py con tus credenciales de FTP:

ftp_host = 'ftp.tuservidor.com'
ftp_user = 'tu_usuario'
ftp_pass = 'tu_contraseña'
ruta_remota = '/ruta/ftp/'

Instalación

  1. Copia el módulo a la carpeta addons de Odoo.
  2. Instalalo desde el panel de Odoo (modo desarrollador).
  3. Crea una acción programada para automatizar la exportación.
