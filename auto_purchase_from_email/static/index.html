Mail Purchase Inbox

Versión: 1.0

Descripción: Este módulo permite la lectura de correos de proveedores con archivos PDF adjuntos y la creación automática de órdenes de compra en Odoo.

Características

  • Lectura automática de correos electrónicos.
  • Procesamiento de archivos PDF con pdfplumber.
  • Creación de órdenes de compra con productos detectados.
  • Emparejamiento automático con contactos de proveedores.

Instalación

  1. Coloca este módulo en tu carpeta de addons.
  2. Reinicia Odoo y actualiza la lista de aplicaciones.
  3. Instala el módulo desde la vista de Apps.
  4. Asegúrate de tener instalado pdfplumber:
pip install pdfplumber

Uso

Los correos serán capturados automáticamente. Desde el registro de entrada, haz clic en "Procesar PDF" para extraer productos y generar la orden de compra.
