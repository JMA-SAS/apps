Stock Deadline Wizard

Modifica la fecha limite de albaranes facilmente

Este modulo permite modificar la fecha limite (scheduled_date) de los albaranes mediante un asistente (wizard).

  • Accede desde el boton en el albaran.
  • Selecciona una nueva fecha desde el wizard.
  • Guarda el cambio facilmente.
